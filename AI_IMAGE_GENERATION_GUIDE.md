## üîß **How to Use Each Platform**

### **Midjourney Setup** (5 minutes)
```
1. Go to midjourney.com
2. Join the Discord server
3. Go to #newbies channel
4. Use /imagine command
5. Paste your prompt
6. Wait for generation (~1 minute)
7. React with üëç to save, or upscale/remix

Command format:
/imagine prompt: [your detailed prompt here]
```

### **DALL-E 3 Setup** (2 minutes)
```
1. Subscribe to ChatGPT Plus ($20/month)
   OR use https://openai.com/dall-e-3
2. Start chat conversation
3. Describe what you want to generate
4. DALL-E 3 generates images automatically
5. Ask for variations or refinements
```

### **Stable Diffusion (Replicate)** (3 minutes)
```
1. Go to replicate.com
2. Sign up with GitHub
3. Search "stable-diffusion"
4. Enter prompt
5. Choose model (SDXL recommended)
6. Set parameters:
   - Guidance Scale: 7-12 (higher = more prompt matching)
   - Steps: 20-50 (higher = better quality, slower)
   - Seed: Random or specific
7. Run and download result
```

## üí° **Pro Tips for Better Results**

### **1. Be Specific About Style**
```
‚ùå Bad: "iOS app mockup"
‚úÖ Good: "Professional iOS app mockup with Material Design 3
         aesthetic, blue primary color, clean minimalist interface,
         iPhone 15 Pro mockup frame, app store quality"
```

### **2. Include Technical Details**
```
‚úÖ Include:
- Resolution (1080x1920, 375x812, etc.)
- Device (iPhone 15, Samsung Galaxy, iPad)
- Design system (Material 3, iOS Human Interface)
- Color scheme (blues, oranges, reds)
- Style (professional, minimalist, sci-fi)
- Lighting (dark background, neon glow)
```

### **3. Reference Real Apps**
```
‚úÖ Good prompts reference:
"Similar to Ingress app but for EVE Online universe"
"Styled like the official EVE Online launcher UI"
"Minimalist like Duolingo but with starmap focus"
```

### **4. Iterate Strategically**
```
Generation 1: Test the concept
Generation 2: Refine based on results
Generation 3: Tweak colors/layout
Generation 4: Request variations
Generation 5: Pick best, enhance in Photoshop
```

### **5. Use Negative Prompts** (Stable Diffusion)
```
Avoid by adding negative prompt:
"Avoid: low quality, blurry, poorly rendered UI,
plastic appearance, broken buttons, distorted text,
amateur design, watermarks"
```
